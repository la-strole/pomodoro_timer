# Pomodoro timer

Simple, lightweight pomodoro timer (<https://en.wikipedia.org/wiki/Pomodoro_Technique>). Watch your tree grow in 25 minutes.

## Docker (temporary command)

1. `docker pull eugeneparkhom/pomodoro` - Listen at 80 hhtp port.
